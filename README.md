## 🚀 Sobre o Projeto

**Dev Galáxias** é um projeto de código aberto (Open Source) focado em revolucionar a forma como aprendemos e praticamos programação. Ele combina a experiência de uma plataforma de cursos online de altíssima qualidade com um **Workspace de Desenvolvimento** integrado direto no navegador.

A ideia nasceu com o objetivo de criar uma "Jornada Estelar" para os alunos, misturando trilhas de ensino fundamentais, exercícios práticos e simulação de ferramentas reais da indústria.

### 🌟 Principais Funcionalidades

- **Dashboard Espacial**: Visão geral do seu progresso, sequências de estudos (streaks) e nível atual.
- **Trilhas de Aprendizado (Tracks)**: Organização de cursos e módulos com progressão visual.
- **Player de Aula Nativo**: Assista às aulas, baixe o material complementar e interaja no fórum de dúvidas na mesma tela.
- **Workspace Lado-a-Lado**: Um ambiente de simulação real que divide a tela entre um **Editor de Código** (para linguagens como Go e TypeScript) e um **Terminal/Preview** que executa o código.
- **Mentor IA**: Integração com Inteligência Artificial para dar dicas em tempo real durante a construção do seu projeto.
- **Gamificação Total**: Sistema de conquistas, ranking estelar, e metas diárias/semanais.

## 🛠️ Tecnologias Utilizadas

Este ecossistema moderno é alimentado por tecnologias de ponta:

- **Frontend**: Next.js (App Router), React 19, Tailwind CSS v4, Lucide Icons, Zustand (Gerenciamento de Estado).
- **Design System**: UI proprietária baseada em Glassmorphism, paleta Dark Mode com contrastes vibrantes e responsividade total.
- **Backend (Planejado/Roadmap)**: Golang (Microsserviços de Alta Performance).
- **Banco de Dados (Roadmap)**: PostgreSQL (Dados Relacionais) e Redis (Cache).
- **Mensageria (Roadmap)**: Apache Kafka para arquitetura orientada a eventos.

## 📖 Como Rodar Localmente (Desenvolvimento)

Siga os passos abaixo para iniciar o **Frontend** na sua máquina:

1. Clone o repositório:
```bash
git clone https://github.com/meusprojetos90/dev-galaxias.git
cd dev-galaxias/frontend
```

2. Instale as dependências:
```bash
npm install
```

3. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

4. Acesse no navegador:
 Abra [http://localhost:3000](http://localhost:3000) e inicie sua jornada!

## 🤝 Contribuindo (Open Source)

O **Dev Galáxias** é orgulhosamente **Open Source**. Nós acreditamos que a educação em tecnologia deve ser acessível, colaborativa e construída pela comunidade, para a comunidade.

Se você deseja contribuir:
1. Faça um *Fork* do projeto.
2. Crie uma branch com sua nova feature: `git checkout -b minha-nova-feature`
3. Salve (commit) as alterações: `git commit -m 'feat: Minha nova feature'`
4. Faça o push para a sua branch: `git push origin minha-nova-feature`
5. Abra um **Pull Request**.

Toda ajuda é bem-vinda: correção de bugs, melhorias no design, traduções, novas telas ou implementação de novas APIs.
